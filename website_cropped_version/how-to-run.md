Steps to run

1. navigate to the website directory

    ```
    cd website
    ```
    
    <br>

2. activate the starter script
    
    For windows :
    ```
    env/scripts/activate
    ```
    For MAC/Linux :
    ```
    source env/bin/activate
    ```

    <br>

3. Run the python file (flask server) : 
    
    For Windows : 
    ```
    python app.py
    ```

    For MAC/Linux :
    ```
    python3 app.py
    ```
